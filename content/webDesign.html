RollingJobs

Se procesó el diseño del proyecto Web “RollingJobs” cuyo objetivo es la postulación de futuros candidatos a las posiciones propuestas en dicha plataforma. La gestión fue planteada inicialmente bajo Adobe XD como maquetación web, Illustrator para edición de ilustraciones que componen la misma y la programación bajo Visual Studio Code con copias de registro de versiones en Git, utilizando React y Bootstrap como lenguajes web.

Box Life Club

Propuesta interna para el microemprendimiento. El objetivo principal es la presentación informativa del emprendimiento y su propuesta de venta de BOX, con accesos a canales de contacto para con los administradores.
